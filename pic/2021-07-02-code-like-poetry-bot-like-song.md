  # 缘起
  短，分行，如诗。  
  这个世界，除了我喜欢的汉字的堆叠，竟然还有这样美妙的存在。  
  不是偶然。  
  是生命中的不期而遇。  
  更是一世的欣喜！  
  所以，偶有闲暇，我就会拾起手指，在短、分行中，感受内心的律动。  
  当然，我本闲散人，怎会成极客？高山仰止，心向往之，于惴惴中前行，算是对自己的一种安慰。
